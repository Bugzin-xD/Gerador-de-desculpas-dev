# 🟢 [Loaden TI](https://loaden.vercel.app) - Dev Excuses Toolbox 💀

> **Status do Projeto**: Em produção (com desculpas prontas para qualquer bug).

Bem-vindo ao **Gerador de Desculpas Dev**! Este projeto foi criado para desenvolvedores que precisam de uma saída elegante (ou caótica) quando o código decide não cooperar. Além do gerador, incluímos a **"Extensão Top"**, que injeta um fundo binário interativo em qualquer site da web.

---

## 🛠️ O que há no pacote?

1.  **Gerador de Desculpas**: Um terminal interativo com efeito *typewriter* e animações de glitch.
2.  **Fundo Matrix Interativo**: Chuva de `0` e `1` com física de explosão vermelha ao clicar.
3.  **Extensão Stealth**: O poder de levar esse visual para o YouTube, Instagram ou qualquer outro lugar.

---

## 🚀 Como Instalar a Extensão

Siga estes passos simples para ativar o modo hacker no seu navegador:

1.  **Baixe os arquivos** presentes na pasta `Extensão top`.
2.  Coloque todos os arquivos (manifest, html, js) em uma **pasta exclusiva** no seu PC.
3.  Abra o seu navegador e vá para o endereço: `chrome://extensions`
4.  No canto superior direito, **ative o Modo do Desenvolvedor**.
5.  Clique no botão **"Carregar sem compactação"**.
6.  Selecione a pasta onde você guardou os arquivos.

**Pronto!** Agora você tem um botão de **Ativar/Desativar** na sua barra de extensões para bugar qualquer site com estilo.
